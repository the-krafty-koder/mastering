# Type command

Used to display a commands type

```
    type <command>
```

# Which

Display an executable's location

```
    which <command>
```

# Help

Get help for shell builtins

```
    help <command>
```

# Man

Display a programs manual page

```
    man <command>
```

# whatis

Display one line documentation for commands

```
    whatis <command>
```

# alias

Used to create your own commands. Command are deleted when the session ends.

```
    alias foo='cd usr; ls; cd -'
```
