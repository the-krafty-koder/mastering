# Ping

`ping linuxcommand.org`
Used to ping a server and receive responses from it

# Traceroute

Lists all the hops a network packet takes to get from a local system to a specified host.

`traceroute slashdort.org`

# Ip

Used to examine a systems network interfaces and routing table
`ip a`

# Netstat

Used to examine various network settings and statistics

`netstat -ie` - examine network interfaces

# Transporting files over a network

-Use `ftp <fileserver>` to connect to a file server

- After connecting and logging in use `get <filename>` to transfer file on server to your machine

# Wget

- Use `wget <file>` to download a file from the internet or a fileserver
