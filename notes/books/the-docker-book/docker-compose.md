# Docker compose

- Used to orchestrate a set of services ( containers that serve different purposes ) in order for them to work together.

# Consul

Used for distributed service discovery

# Swarm

Used for orchestration and clustering of docker.

# Commands

`docker-compose up` - used to run the compose file
`docker-compose stop` - used to stop docker-compose services.
`docker-compose up -d` - run docker-compose services while detached.
`docker-compose ps` - list the docker-compose services
`docker-compose start` - restart docker-compose services stopped by the stop command.
